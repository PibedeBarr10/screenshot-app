## Getting Started

### Prerequisites

Install the libraries with the command:

```sh
pip install -r requirements.txt
```
or:
```sh
python -m pip install -r requirements.txt
```

### Usage
1. Clone the repo:
```sh
git clone https://github.com/PibedeBarr10/screenshot-app.git
```
2. Open the app:
```sh
python app.pyw
```
